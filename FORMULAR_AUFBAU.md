## Punktesystem (als Abschnitts-Beschreibung einfügen)

```
GRUPPENPHASE:
  Richtiges Ergebnis        → 4 Punkte
  Richtige Tordifferenz     → 3 Punkte
  Richtige Tendenz/Sieger   → 2 Punkte

KO-RUNDEN (nur Weiterkommen):
  Sechzehntelfinale         →  3 Punkte je Team  (NEU bei WM 2026)
  Achtelfinale              →  5 Punkte je Team
  Viertelfinale             → 10 Punkte je Team
  Halbfinale                → 15 Punkte je Team
  Finale (beide korrekt)    → 20 Punkte
  Weltmeister               → 25 Punkte
```

## Formular-Einstellungen

- **Antworten:** Einmal pro Person (verhindert Mehrfacheinreichung)
- **Startdatum/Ablaufdatum:** Am 11.06.2026 um 13:00 Uhr (Anpfiff erstes Spiel) deaktivieren
- **Ergebnisse anzeigen:** NEIN (niemand soll Tipps anderer sehen)
- **Link teilen:** Über "Freigeben" → Link generieren → per WhatsApp/Email verteilen

---

## Workflow nach Tippabgabe

1. forms.office.com → Dein Formular → **"Antworten" → "In Excel öffnen"**
   → Lädt `WM Tippspiel(1.0).xlsx` aus OneDrive herunter
2. Lokal speichern als `Tipps_Responses.xlsx`
3. `Ergebnisse.xlsx` mit aktuellen Spielergebnissen befüllen
4. In Terminal/Eingabeaufforderung (im Ordner "Fußball WM 2026"):
   ```
   python auswertung.py Tipps_Responses.xlsx Ergebnisse.xlsx
   ```
5. `Rangliste_[Datum].xlsx` per WhatsApp/Email an alle verschicken
